Pro Version Getting Started

License Server

You will need to run a license server in order to use the Pro version. See License Server for details.

Minimum Server Count

If you have more than one replica, you must have at least three servers. This doesn't mean you have to have three replicas. You must either have at least three replicas or have at least two shards with two replicas. If you truly only want one shard with two replicas, you will need to run two servers on each box.

Configuration

With the Pro version, you may add replicas to your shards. The following example partial configuration shows how to create a cluster with two shards, each having three replicas.

    {
        "dataDirectory": "$HOME/db-data",
        "installDirectory": "$HOME/sonicbase",
        ...
        "shards" : [
            {
                "replicas": [
                    {
                        "privateAddress": "localhost",
                        "publicAddress": "localhost",
                        "port": 9010
                    },
                    {
                        "privateAddress": "localhost",
                        "publicAddress": "localhost",
                        "port": 9020
                    },
                    {
                        "privateAddress": "localhost",
                        "publicAddress": "localhost",
                        "port": 9030
                    }
                ]
            },
            {
                "replicas": [
                    {
                        "privateAddress": "localhost",
                        "publicAddress": "localhost",
                        "port": 9040
                    },
                    {
                        "privateAddress": "localhost",
                        "publicAddress": "localhost",
                        "port": 9050
                    },
                    {
                        "privateAddress": "localhost",
                        "publicAddress": "localhost",
                        "port": 9060
                    }
                ]
            }
        ]
    }
