Startup

Start the client by typing, from the bin subdirectory of the install directory, "./cli" for mac or linux and "./cli.bat" for cygwin and "cli.bat" for Windows.

Plain Client

If you are having problems with the colors, text positiong or windows resizing, you may wan to use the plain client. To run the plain client, add "--plain" to the cli command.

Use a Cluster

Type "use cluster <cluster name>". The cluster name must match a config file in the config directory. The name of the config file must follow "config-<cluster name>.json".

Deploy Cluster

To deploy the image to the servers in the cluster, type "deploy cluster". Note that you must setup passwordless ssh between all the servers.

Start Cluster

To start a cluster type "start cluster".

Purge Cluster

To purge a cluster type "purge cluster". This will purge and stop the cluster.

Create a Database

Type "create database <database name>".

Use a Database

To use a database type "use database <database name>".
