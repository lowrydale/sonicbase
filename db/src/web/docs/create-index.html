SQL Create Index Statement

Creates an index for the specified table

Syntax

CREATE [UNIQUE] INDEX index_name on table_name(column [,column...])
index_name:  name of the index to create
table_name:  name of the table to create the index in
column:   name of the column to include in the index
UNIQUE  If included, the index contents must include unique keys

Example


	create index age on persons(age)
