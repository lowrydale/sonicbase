Creates a table to store records in.

Syntax

CREATE TABLE table_name (column type [,column type...] [PRIMARY KEY (key_column [,key_column...])
table_name:  name of the table to create
column:   name of the column to create
type:   data type for the column to create. See Data Types
key_column: name of the column to include in the primary key


Example:

create table persons (name VARCHAR, age INT, ssn VARCHAR, PRIMARY_KEY(ssn))
