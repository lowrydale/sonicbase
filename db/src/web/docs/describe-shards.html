SQL Describe Shards Statement

Describes the shards. Sizes of the indices in each shard are returned. Also the keys that define the boundaries for each shard are returned

Syntax

DESCRIBE SHARDS

Example

PreparedStatement stmt = conn.prepareStatement("describe shards")
ResultSet rs = stmt.executeQuery();
while (rs.next()) {
  System.out.println(rs.getString(1))
}
