SQL Describe Tables Statement

Lists the tables in the current database.

Syntax

DESCRIBE TABLES

Example


PreparedStatement stmt = conn.prepareStatement("describe tables")
ResultSet rs = stmt.executeQuery();
while (rs.next()) {
    System.out.println(rs.getString(1));
}
