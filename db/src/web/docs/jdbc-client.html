Prerequisites

Before you connect to the database you need to create the database using the client. See the Getting Started Guide for an example.

You must also include the SonicBase jdbc driver jar in your classpath. The jar is located int the "lib" directory of the install directory. It is named "sonicbase-[version].jar".

JDK Requirements

The jvm running the jdbc client needs to be at least version 7.

Connecting

Include the following line before creating a connection:
    Class.forName("com.sonicbase.jdbcdriver.Driver");

Connect to the database with the following code:
    Connection conn = DriverManager.getConnection("jdbc:sonicbase:<address>:9010/<database name>");
