Monitoring

To monitor servers, type "describe server stats" in the SonicBase client. Stats include the following:
  • CPU - total cpu utilization by the process
  • ResMem - total resident memory for the process
  • JMemMin - min java heap memory used
  • JMemMax - max java heap memory used
  • NetIn - inbound network traffic
  • NetOut - outbound network traffic
  • Disk - amount of freed disk space on the mount point where the database is stored

You must install the "ifstat" package on Mac OS X and "dstat" on Linux in order to monitor network traffic.
