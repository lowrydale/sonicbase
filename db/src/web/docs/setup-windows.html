PowerShell

Powershell scripting must be enabled. Also, remote powershell commands must be enabled. To enable these features open a powershell session on the server and on the client as an administrator and type the following:

  Enable-PSRemoting -Force
  Set-ExecutionPolicy RemoteSigned

Credentials

When you connect to a cluster for the first time, SonicBase will ask you for credentials for your servers. These credentials are stored in an encrypted file on the client and are subsequently used when interacting with the servers.
