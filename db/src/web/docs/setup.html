JDK Requirements

The jvm running the database server and the jdbc client needs to be at least version 7.

Python Requirements

Python must be installed on the clients and the servers

Operating System Requirements

SonicBase only runs on Linux and Mac OS X.

Server Memory Size

Size the server memory at approximately 2x the records size for small records (150 bytes). Allocate a slightly less percentage for larger records.

Disk Size

Size the disk at 3x the server memory size plus enough for disk-based result sets (perhaps another 3x).

Config File

You must create a config file for your cluster. See config for details.

Port Access

The servers must be able to communicate with one another on the configured ports in the config file. Also, the client needs to communicate with each server on the configured ports. If you don't want access to these port outside a firewall, move the client inside the firewall and run it from there.

SSH Access

There must be passwordless ssh access between all the servers and between the client and all the servers. If you don't want ssh access to the servers outside a firewall, move the client inside the firewall and run it from there.
