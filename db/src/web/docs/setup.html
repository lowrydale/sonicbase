JDK Requirements

The jvm running the database server needs to be at least version 7.

Server Memory Size

Size the server memory at approximately 2x the records size for small records (150 bytes). Allocate a slightly less percentage for larger records.

Disk Size

Size the disk at 3x the server memory size plus enough for disk-based result sets (perhaps another 3x).

Disk Optimization

An optimized disk will speed up server start time. It is recommended that you have an SSD disk with 5k iops.

Config File

You must create a config file for your cluster. See config for details.
