SonicBase is a free in-memory distributed relational database. You can scale SonicBase to billions of records. See the documented benchmarks for more details. SonicBase is cross-platform, supporting Linux, Mac, Cygwin and Windows. The primary interface for SonicBase is JDBC. Sonicbase supports cross-shard transactions and cross-shard joins. There is an admistration client that can be used to deploy clusters, start clusters, stop clusters and execute sql commands. SonicBase is durable so you won't loose any changes when restarting the cluster. SonicBase has near-linear scaling ability.
