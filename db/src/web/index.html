Free Database

A highly functional subset of features is provided in the free version of SonicBase. Additional features are available for a modest fee.

In-Memory Database

SonicBase stores all data in memory. It uses a mix of on-heap and off-heap memory. Additionally, disk-based snapshots and logs are used to achieve durability.

Distributed Database

Records and indices are spread out across as many servers as you deem necessary. The data is transparently divided into shards and it is dynamically distributed as you add or delete records. You can scale SonicBase to billions of records. See the documented benchmarks for more details. Sonicbase supports cross-shard transactions and cross-shard joins.

Relational Database

SonicBase supports the SQL syntax. SQL support is accessed via the JDBC driver.

Cross-platform Database

SonicBase is cross-platform, supporting Linux, Mac, Cygwin and Windows.
