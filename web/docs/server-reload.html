Reload Commands

Issue the following reload commands from the administrative client.

Reload Server

To reload a server from a peer, type "reload server <shard> <replica>, where shard and replica specify the server you wish to reload. You would need to take this action in the case you are replacing a bad server or disk where the database resides.

Reload Server Status

To get the status of a server reload, type "reload server status <shard> <replica>, where shard and replica specify the server you are reloading. Returns "complete" or "running" depending on the state of the reload.

Reload Replica

To reload a replica from its peers, type "reload replica <replica>, where replica specifies the replica you wish to reload. You would need to take this action in the case you are adding a new replica.

Reload Replica Status

To get the status of a replica reload, type "reload replica status <replica>, where replica specifies the replica you are reloading. Returns "complete" or "running" depending on the state of the reload.
